# 機種内訳のLINE共有機能追加 完了報告

LINE共有メッセージの統計セクションに、機種内訳（どの機種を何台打ったか）を追加しました。

## 変更内容

### [share.js](file:///h:/gravity/pachinko-ev-calculator/share.js)

- **機種内訳の集計**: 貸玉レートごとの統計ループ内で、そのレートに含まれる機種名と台数を集計するロジックを追加しました。
- **古い順での表示**: `shareData` が最新順（unshiftで追加されるため）であることを考慮し、`unshift` を使って機種名を収集することで、内訳を古い順に表示するように調整しました。
- **メッセージへの挿入**:
  - **詳細モード**: 日時表示の直後に `機種1 (n台) / 機種2 (m台)` の形式で挿入。
  - **簡略モード**: 統計見出しの前に `機種内訳: 機種1 (n台) / 機種2 (m台)` の形式で挿入。

## 検証結果

### 送信されるメッセージの形式（イメージ）

#### 詳細モード

```text
2026/02/27 16:30
大海5 (2台) / 北斗10 (1台)
【4円】総投資/...
```

#### 簡略モード

```text
2026/02/27 16:30
機種内訳: 大海5 (2台) / 北斗10 (1台)
【4円 統計】
💰 合計期待値: ...
```

以上の修正により、統計データの内訳がより明確に共有できるようになりました。
