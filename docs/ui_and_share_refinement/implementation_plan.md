# UIと共有機能の微調整 実装計画

LINE共有時のテキストレイアウトを整理し、アプリ内の履歴表示（詳細モード）を見やすく調整します。

## Proposed Changes

### [共有機能]

#### [MODIFY] [share.js](file:///h:/gravity/pachinko-ev-calculator/share.js)

- 複数選択時の統計セクションにおいて、各レートごとの境界線（`----`）を整理します。
- 統計セクション全体と、その後に続く「個々のデータ」セクションの間に明確な区切り線を追加します。

### [UI表示]

#### [MODIFY] [ui-manager.js](file:///h:/gravity/pachinko-ev-calculator/ui-manager.js)

- 履歴表示の「詳細」モードにおいて、個々のデータのテキストを中央寄せにします。
- データの左右に適切な余白（パディング）を追加し、視認性を向上させます。

---

## Verification Plan

### Automated Tests

- なし（UI/テキストの微調整のため）

### Manual Verification

1. アプリを起動し、履歴データを複数選択して「LINE共有」をクリックします。
2. 生成されたテキストを確認し、統計と個々のデータの間に区切り線が入っていること、統計内の余分な区切り線が解消されていることを確認します。
3. 履歴一覧で「詳細表示」に切り替え、個々のデータが中央に寄っており、左右に余白があることを確認します。
