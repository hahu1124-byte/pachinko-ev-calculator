# 履歴表示レイアウトの修正（余白の確保）

履歴表示の詳細モードにおいて、前回の修正で「センター寄せ」になっていた箇所を修正し、ユーザーの要望通り「左右に余白を設ける」形に変更します。

## 提案される変更

### UIマネージャー

#### [MODIFY] [ui-manager.js](file:///h:/gravity/pachinko-ev-calculator/ui-manager.js)

- `renderHistory` 関数内の詳細モード描画ロジックを修正。
- `div.style.textAlign = 'center'` を削除。
- `justify-content: center` を削除し、デフォルトの `space-between`（CSSで定義済み）または適切な配置に戻す。
- `padding` を調整し、左右に十分な余白（例: `0.75rem 1.25rem`）を確保する。

## 検証プラン

### 手動確認

- ブラウザでパチンコ期待値計算ツールを開く。
- 履歴を数件保存し、「詳細」表示モードにする。
- 各履歴アイテムがセンター寄せになっておらず、左右に適切な余白があることを確認する。
