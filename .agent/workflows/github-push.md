# GitHubプッシュワークフロー

// turbo-all

1. 修正箇所のコード内に、処理の意図や内容を説明する日本語コメントが過不足なく追記されているか確認する
2. `git add .` を実行して変更をステージングする
3. `git commit -m "アップデート内容"` の形式でコミットする
4. `git push` を実行して、`https://github.com/hahu1124-byte/pachinko-ev-calculator` リポジトリに変更を反映させる
